## Overview

This project investigates whether fine-tuned transformer-based models can detect refugee- and migrant-related misinformation on social media platforms. The work was conducted in collaboration with the United Nations High Commissioner for Refugees (UNHCR).

The pipeline has two phases:

**Phase 1 — Benchmarking:** Fine-tuning four transformer models on a compiled labeled misinformation dataset and evaluating on a manually annotated UNHCR validation set.

**Phase 2 — Distillation:** Using the best Phase 1 model as a teacher to distill knowledge into a smaller ModernBERT-large student model via a step-by-step distillation approach.

Best distilled model: **accuracy 0.772, F1-score 0.832** on the UNHCR validation set — comparable to the teacher at roughly 1/20th the size.

## Setup

```bash
pip install -r requirements.txt
python -m spacy download en_core_web_sm
```

You'll need access to gated models on HuggingFace (LLaMA, Gemma). Set your token:

```bash
export HF_TOKEN=your_token_here
```

Place your dataset CSV files in `data/raw/` (see `data/README.md` for details).

## Usage

### Quick start (full pipeline)

```bash
python app.py pipeline --model llama --dataset twitter_filtered
```

### Step by step

**1. Preprocess data**
```bash
python scripts/run_preprocessing.py --config twitter_filtered --balance
python scripts/run_preprocessing.py --all  # all 4 configs
```

**2. Phase 1 — Fine-tune a model**
```bash
python scripts/run_phase1.py --model llama --dataset twitter_filtered
python scripts/run_phase1.py --model llama --dataset twitter_filtered --hp-search
python scripts/run_phase1.py --model all --dataset all  # full benchmark
```

**3. Phase 2 — Distillation**
```bash
python scripts/run_phase2.py \
    --teacher-path models/llama_twitter_filtered \
    --dataset twitter_filtered \
    --domain-adapt

# With hyperparameter search
python scripts/run_phase2.py \
    --teacher-path models/llama_twitter_filtered \
    --dataset twitter_filtered \
    --hp-search --domain-adapt
```

**4. Evaluate**
```bash
python scripts/run_evaluation.py \
    --model-path models/distilled_modernbert \
    --model-type distilled \
    --dataset urxd \
    --shap --error-analysis
```

## Models

| Model | Base | Size | Technique |
|---|---|---|---|
| LLaMA 3.1 | meta-llama/Llama-3.1-8B-Instruct | 8B | LoRA + 4-bit |
| Gemma2 | google/gemma-2-9b-it | 9B | LoRA + 4-bit |
| ModernBERT | answerdotai/ModernBERT-large | ~400M | LoRA (encoder) |
| DeepSeek | deepseek-ai/DeepSeek-R1-Distill-Llama-8B | 8B | LoRA + 4-bit |

The distillation student is ModernBERT-large — same architecture as Phase 1, but trained via knowledge distillation from the best teacher model.

---

## Datasets

**Labeled Misinformation Dataset (LMD):** Compiled from multiple public misinformation datasets (FakeNewsNet, LIAR, MultiFC, etc.) covering news articles and social media posts. Used for training and testing. Available in 4 configurations.

**UNHCR Refugee X Dataset (URXD):** 250 X (Twitter) posts about refugees/migrants, manually annotated by the authors in collaboration with UNHCR. Used as the out-of-domain validation set.

See `data/README.md` for more details and instructions on obtaining the data.

---

## Results

Phase 1 best model: LLaMA 3.1 on `twitter_filtered` with feature engineering (F1: 0.954 on test set).

Phase 2 distilled ModernBERT-large:
- Accuracy: 0.772
- F1-score: 0.832
- ~1/20th the size of the teacher model

See `notebooks/03_results_analysis.ipynb` for full results and SHAP analysis.
